API Documentation

API Reference

POST /api/auth/login

Authenticate a user and return a JWT token.

Request Body:
{
  "username": "string",
  "password": "string"
}
Response:
{
  "token": "string",
  "expires_in": 3600
}

GET /api/users

Retrieve a list of users.

Headers:
Authorization: Bearer {token}
Response:
{
  "users": [
    {
      "id": 1,
      "name": "John Doe",
      "email": "john@example.com"
    }
  ]
}

POST /api/reports/generate

Generate a new report.

Headers:
Authorization: Bearer {token}
Request Body:
{
  "type": "string",
  "parameters": {
    "start_date": "2023-01-01",
    "end_date": "2023-12-31"
  }
}
Response:
{
  "report_id": "string",
  "status": "processing"
}
